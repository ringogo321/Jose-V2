# Jose V2 Demo

Simple static demo page for a high-volume recruiting AI workflow.

## Files
- `index.html`
- `recruiter-questions.html`
- `styles.css`
- `script.js`

## Run locally
Open `index.html` in your browser.

## Deploy (GitHub Pages)
1. Push this folder to a GitHub repo.
2. In GitHub, open `Settings` → `Pages`.
3. Under `Build and deployment`, set `Source` to `Deploy from a branch`.
4. Set `Branch` to `main` (or `master`) and `/root`.
5. Click `Save` and wait for the site URL to appear.

## Custom domain
1. Create a file named `CNAME` in the repo root with your domain, for example:
   ```txt
   careers.example.com
   ```
2. In `Settings` → `Pages`, set the custom domain to match the `CNAME` value.
3. Add the required DNS records at your domain host (GitHub will show the exact values).
